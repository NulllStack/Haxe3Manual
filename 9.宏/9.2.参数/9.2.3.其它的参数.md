# 9.2.3.其它的参数

如果一个宏最后的参数是 Array类型，宏可以以数组形式接受一个任意数量的额外参数：

```haxe
import haxe.macro.Expr;

class Main {
  static public function main() {
    myMacro("foo", a, b, c);
  }

  macro static function myMacro(e1:Expr, extra:Array<Expr>) {
    for (e in extra) {
      trace(e);
    }
    return macro null;
  }
}
```
