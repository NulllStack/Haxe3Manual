# 10.2.6.Option

一个 Option 是Haxe标准库中的一个枚举，如下形式定义：

```haxe
enum Option<T> { 
    Some(v:T); 
    None; 
} 
```

它可以被使用在各种各样的状况，比如 沟通一个方法是否有一个有效的返回，如果是的话，它返回什么值：

```haxe
import haxe.ds.Option; 

class Main { 
    static public function main() { 
        var result = trySomething();
        switch (result) { 
            case None:
                trace("Got None"); 
            case Some(s): 
                trace("Got a value: " +s); 
        } 
    } 
    
    static function trySomething():Option<String> { 
        if (Math.random() > 0.5) { 
            return None; 
        } else { 
            return Some("Success"); 
        }
    }
}
```
