# 10.2.3.列表

列表是一个存储的元素的集合。从表面上看，一个列表类似于一个数组。然而，潜在的实现非常不同。这体现在一些功能差异：

- 列表不能被使用方括号索引，即 [0] 形式。
- 列表不能被初始化。
- 没有列表推导。
- 列表在迭代的时候可以自由修改和删除元素。

使用列表的一个简单例子：

```haxe
class Main { 
    static public function main() {
        var myList = new List<Int>(); 
        for (ii in 0...5) 
            myList.add(ii);
        trace(myList); //{0, 1, 2, 3, 4}
    }
} 
```
